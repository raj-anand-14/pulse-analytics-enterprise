## Overview

Pulse Analytics Enterprise is a premium analytics platform built using **Google Apps Script**, **HTML5**, **CSS3**, and **Vanilla JavaScript**.

The application provides:

- Executive Dashboard
- Cluster Analytics
- Team Performance
- Employee Analytics
- Bronze / Silver / Gold Ranking
- Live CSV Integration
- Reports
- Export
- AI Ready Architecture
