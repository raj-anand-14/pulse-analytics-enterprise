# Phoenix Manifesto

> Built with discipline. Designed for excellence.

Phoenix is not just another dashboard.

It is an engineering exercise in building enterprise-grade software on top of Google Apps Script.

Every module is intentional.
Every dependency is justified.
Every commit improves the project.
Every release tells a story.

We optimize for maintainability over shortcuts.

We engineer software that we are proud to publish.

If a feature cannot meet this standard, it waits until it can.
